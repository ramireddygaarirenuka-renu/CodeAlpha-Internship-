## 🗂️ Completed Tasks

### 🎯 Task 1: Hangman Game
A console-based word guessing game where:
- The user guesses one letter at a time
- Limited incorrect attempts are allowed
- The word progress is displayed after each guess
- The game ends when the word is guessed or attempts are exhausted

**Concepts Used:**
- `random` module
- While loops
- Conditional statements
- Strings & Lists

---

### 📊 Task 2: Stock Portfolio Tracker
A simple portfolio calculator that:
- Accepts stock names and quantities from the user
- Uses predefined stock prices
- Calculates total investment value
- Displays a portfolio summary

**Concepts Used:**
- Dictionaries
- User input handling
- Arithmetic operations
- Loops

---

### 🤖 Task 3: Basic Chatbot
A rule-based chatbot that:
- Responds to user greetings
- Handles simple conversational inputs
- Exits when the user types "bye"

**Concepts Used:**
- If-elif statements
- String methods
- Infinite loops with break condition
